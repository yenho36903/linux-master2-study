# 1과목 리눅스 운영 및 관리
1.파일 시스템 관련 명령어 
1.1 권한 및 그룹 설정
모든 파일과 디렉터리에는 접근권한,소유권가 부여됨 
소유권 
파일 혹은 디렉터리에 대한 사용자와 그룹들의 소유 권한
관련 명령어
 chown [옵션] 소유자[:그룹명] 파일명 
 사용자 소유권,그룹 소유권 변경
 chgrp [옵션] 그룹명 파일명 
 그룹 소유권 변경 
 옵션
 -R : 하위 디렉터리를 포함 전체 변
허가권 
ls -l -> 파일유형과 허가권 알수있음
- 파일사용자 : 소유자,그룹,기타 사용자
- 권한: 읽기(r),쓰기(w),실행(x)
- 명령어
 chmod (파일,디렉터리의 접근 허가권 변경)
 옵션 : - R : 하위 디렉터리를 포함해서 모든 디렉터리 및 파일의 권한 변경
 숫자로 바꾸기 : r은 4 , w 은 2 , x은 1
 ex: rw- : 42- , r-- : 4--, rwx : 421
기호로 바꾸기 :
사용자 (u : 소유자, g : 그룹 , o : 기타사용자 , a : 모든 사용자 u+g+o
연산자 (+ : 허가권 부여, - : 허가권 제거 , = : 특정 사용자에게 허가권 지정 )
권한 (읽기(r),쓰기(w),실행(x))
umask (기본 허가권)
- S : 문자로 표기
파일 기본권한 : 666 , 디렉터리의 기본권한 : 777
